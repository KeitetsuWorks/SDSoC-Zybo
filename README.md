Zybo Zynq-7000 SDSoC Platform
========


## Description

This Custom SDSoC Environment Platform is created for SDx 2017.4.1, Vivado 2017.4.1, and PetaLinux 2017.4.


## Requirement

### Hardware

* [Zybo Zynq-7000 ARM/FPGA SoC Trainer Board](https://reference.digilentinc.com/reference/programmable-logic/zybo/start)


### Software

* Xilinx PetaLinux 2017.4
* Xilinx SDSoC 2017.4.1
* Xilinx Vivado 2017.4.1
